# IR-Assignment-2

Assignment-2 for CS F469 Information Retrieval Course

# How to run

- Ensure corpus files are extracted into `dataset` folder.
- Ensure `python3` and `pipenv` are installed.
- `pipenv shell`
- `pipenv install`
- `python3 app.py`
- Open the sever in your browser and enter your query.
